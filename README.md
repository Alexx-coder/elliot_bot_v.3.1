# elliot_bot_v.3.1

Это Бот Эллиот
Текущие возможности(до новой версии):
-5 функциональных модулей
-Интерактивное меню выбора
-Консольный интерфейс
-Обработка некорректного ввода
-Класс ошибок
-Ответы на вопросы пользователя
-Тематические разлелы к каждой функции

 Модули:
1.Математика — решение задач и вычисления
2.Помощь по Python — помощь в программировании
3.Советы по работе с компьютером — оптимизация и исправления
4.Командная строка — утилиты CMD/PowerShell
5.Установка ОС — руководства по Windows/Linux

Классы ошибок:
1.ElliotBotError - базовый класс
2.InvalidCommandError - для неизвестных боту команд
3.NotFoundFunctionError - для ненайденных функций
4.ValidationError - для некорректного ввода
5.MathError-связан с первой функции(Математика)
